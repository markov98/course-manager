# Course Manager

A full-stack web application built as part of an interview task.  
- **Frontend**: Angular  
- **Backend**: Express (Node.js)  

---

## Getting Started

```bash
git clone https://github.com/markov98/course-manager.git
cd course-manager
docker-compose build
docker-compose up
```

Access the app at localhost:80
